# GHOST//NET : 404

A custom 404 error page inspired by the 1995 film Ghost in the Shell.

## The Concept

Your request propagates through six layers of the net and is intercepted
by Project 2501 — a sentient program that emerged spontaneously from the
complexity of global data traffic. The page you were looking for does not
exist. But something that exists without a body noticed your search arriving.

Navigate the classified Section 9 node network to establish a route to
Layer 6. Four nodes unlock the entity. The terminal runs the whole time.
